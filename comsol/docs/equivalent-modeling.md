# COMSOL 全耦合等效建模说明

## 背景

标准 **固体力学 + 传热** 耦合通常将温度作为输入，计算热膨胀变形，难以直接得到“结构变形引起温度场变化”的反馈。

本项目采用的等效策略（与热-磁-电-弹性板壳全耦合理论一致）：

1. 在 **压电** 接口中定义材料；
2. 将热应力模量 λ 映射为等效压电耦合系数；
3. 将体积热容 c 映射为等效介电常数；
4. 求解得到的**电场分布**按理论关系解释为结构内的**温差** θ。

## 建模要点

- 板厚方向离散为与 MATLAB 相同的层数（默认 10 层）；
- 每层在中面高度处取 FG 等效参数（与 `materials/build_layer_materials.m` / `generate_cases.py` 一致）；
- 电极间距 `hE`、层厚 `zC1/zC2` 与输入文件 MATERIAL 段一致。

## 验证判据

对同一工况、同一对比点：

```
Δ(%) = |R_MATLAB - R_COMSOL| / |R_COMSOL| × 100%  < 5%
```

其中 R 可为挠度 w 或厚度方向温差 θ。
